# Watermarking_Research_Project

## How to run the program 

 ```
python watermarking_multiple3.py
```
Please install the below packages. They are:
  * glob3==0.0.1
  * numpy==1.17.4
  * opencv-python==4.1.2.30
  * Pillow==6.2.1
  * PyWavelets==1.1.1
